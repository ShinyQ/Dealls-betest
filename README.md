# Kurniadi - BE Test

## Stack :
```
1. NodeJS
2. ExpressJS
2. MongoDB
3. Redis Server

- Web App : vercel
- MongoDB : cloud.mongodb.com
- Redis : aiven.io
```

## Installation :
```
1. Copy ".env.example" to make ".env"
2. Fill the config according to your machine
3. npm install
```

### Run :
```
npm start
```

### Test :
```
npm test
```
